# Rule-Based Intelligent Agent for Stock Trading Decisions

## Project Description

This project is a simple Python program that makes stock trading decisions using fixed rules.

The program reads stock price data from a CSV file, calculates daily percentage change, and gives one of three decisions:

* BUY
* SELL
* HOLD

It also calculates simple profit and saves the result in a CSV file. A graph is displayed to show stock price movement along with trading decisions.

## Files Included
* `main.py` → Main Python program
* `stock_data.csv` → Input stock price data
* `output_table.csv` → Output generated by the program
* `README.md` → Project documentation
* `trading_graph.png` → Visualised data
* `Intelligent Trading Agent.pdf` → Project report

## Input Format

The input file must be named `stock_data.csv`.

The file should contain two columns:

* `Day`
* `Price`

Example:

```csv
Day,Price
1,100
2,98
3,97
4,99
5,102
```


## Decision Rules
The program calculates percentage change in stock price using daily values.
Rules used:
* If price change is less than or equal to -2%, decision = BUY
* If price change is greater than or equal to 2%, decision = SELL
* Otherwise, decision = HOLD

## Output Generated
The program produces:

### Terminal Output
Shows day-wise stock price, percentage change, decision, and profit.

### CSV Output
Creates a file named `output_table.csv`.

### Graph Output
Displays stock price graph with BUY and SELL points.

## How to Run
1. Keep all project files in the same folder:
   * `main.py`
   * `tradingagent.py`
   * `stock_data.csv`

2. Run the main program to generate trading decisions and output table:

```bash
python main.py
```

3. The program will create:

   * terminal output
   * `output_table.csv`

4. To display the graph separately:

```bash
trading_graph.png
```

5. The graph will show stock price movement with BUY and SELL points.
